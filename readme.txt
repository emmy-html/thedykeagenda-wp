== Description ==

This theme is the parent theme for The Lez Agenda. It was created using the lezagenda HTML5 WordPress boilerplate, and the custom HTML, CSS & PHP of Emmy. Eventually, it'll contain a child theme with a day mode, but for now, night mode is the preferred layout.
